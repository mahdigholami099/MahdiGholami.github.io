Valock

I've created a program for the instance lock of agents in Valorant, the popular game where players need to choose and lock their preferred agent before a match begins. This program simplifies the agent selection process by automating the locking mechanism. With my fan-made project, you can ensure that your desired agent is locked in quickly and efficiently, allowing you to focus on strategizing and maximizing your gameplay experience.
